The demo project uses the CocoaPods method by default. If you want to use SPM (Swift Package Manager), please add the folder containing the current Package.swift file to the demo project Package Dependencies.  Additionally, you will need to include the SDWebImage and SSZipArchive libraries.

Package Dependencies location
../../Pods-NvShortVideoEdit
https://github.com/SDWebImage/SDWebImageWebPCoder.git
https://github.com/ZipArchive/ZipArchive.git
