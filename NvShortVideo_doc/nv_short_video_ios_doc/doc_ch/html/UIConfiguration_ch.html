短视频UI配置项

前言 

1、界面上的UI控件，可以进行定制，比如替换图标、改变颜色等。
2、每个页面会标注出可配置的元素，红色文字代表UI控件的key，黄色文字代表UI配置项的类
3、使用时可以用内部定义的好的字符串类型
4、如果需要替换图标,需要先在您自己的工程中添加好图标

拍摄UI配置项









使用示例：

ios 

//以”capture_capture_close_bt“为例
NvButtonTheme *buttonTheme = [[NvButtonTheme alloc] init];
buttonTheme.imageName = @"homepage_logo";
self.videoConfig.captureConfig.customTheme[NvCaptureCloseBtKey] = buttonTheme;

//以”capture_duration_label“为例
NvLabelTheme *labelTheme = [[NvLabelTheme alloc] init];
labelTheme.textColor = UIColor.redColor;
self.videoConfig.captureConfig.customTheme[NvCaptureDurationLabelKey] = labelTheme;

//以”capture_music_menu_view“为例
NvViewTheme *viewTheme = [[NvViewTheme alloc] init];
viewTheme.backgroundColor = UIColor.redColor;
self.videoConfig.captureConfig.customTheme[NvCaptureMusicMenuViewKey] = viewTheme;

//以”capture_capture_record_bt_set“为例
NvRecordBtTheme *record = [[NvRecordBtTheme alloc]init];
record.minimumTrackTintColor = UIColor.redColor;
self.videoConfig.captureConfig.customTheme[NvCaptureRecordBtSetKey] = record;

其他类型的使用可以参考相关的接口文档

anroid 

                NvVideoConfig videoConfig = new NvVideoConfig();
                NvCaptureConfig captureConfig = new NvCaptureConfig();
                videoConfig.setCaptureConfig(captureConfig);
                Map<String, Object> customTheme = new HashMap<>();
                captureConfig.setCustomTheme(customTheme);
                //以”capture_capture_close_bt“为例
                NvButtonTheme  buttonTheme = new NvButtonTheme();
                buttonTheme.setImageName("your_custom");
                customTheme.put(NvKey.NvCaptureCloseBtKey, buttonTheme);

                //以”capture_duration_label“为例
                NvLabelTheme labelTheme = new NvLabelTheme();
                labelTheme.setTextColor("#FFFFFF");
                customTheme.put(NvKey.NvCaptureDurationLabelKey, labelTheme);

                 //以”capture_music_menu_view“为例
                NvViewTheme viewTheme = new NvViewTheme();
                viewTheme.setBackgroundColor("#000000");
                customTheme.put(NvKey.NvCaptureMusicMenuViewKey, viewTheme);

                //以”capture_capture_record_bt_set“为例
                NvRecordBtTheme record = new NvRecordBtTheme();
                record.setMinimumTrackTintColor("#FF0000");
                customTheme.put(NvKey.NvCaptureRecordBtSetKey, record);

其他类型的使用可以参考相关的接口文档

flutter 

//以”capture_capture_close_bt“为例
var buttonTheme = NvButtonTheme();
buttonTheme.imageName = "homepage_logo";

//以”capture_duration_label“为例
var labelTheme = NvLabelTheme();
labelTheme.textColor = "#FF0000";

//以”capture_music_menu_view“为例
var viewTheme = NvViewTheme();
viewTheme.backgroundColor = "#FF0000";

//以”capture_capture_record_bt_set“为例
var record = NvRecordBtTheme();
record.minimumTrackTintColor = "#FF0000";

var customTheme = {
  NvCaptureCloseBtKey: buttonTheme,
  NvCaptureDurationLabelKey:labelTheme,
  NvCaptureMusicMenuViewKey:viewTheme,
  NvCaptureRecordBtSetKey:record
};

videoConfig.captureConfig.customTheme = customTheme;
